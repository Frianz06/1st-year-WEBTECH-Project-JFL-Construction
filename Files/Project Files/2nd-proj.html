MINI-GROCERY & DENTAL CLINIC PROJECT


PROJECT DESCRIPTION


This project converted a vacant retail location into a functioning dental clinic and mini-grocery store.
We built constructed walls to divide the dental clinic and grocery store. The clinic included a distinct operating room
and a bathroom, which were all built with wall divisions. Floor tiles were used throughout the room to provide the dental clinic
and grocery store with a clean and professional appearance. The client was greatly satisfied with the end result of the project.
