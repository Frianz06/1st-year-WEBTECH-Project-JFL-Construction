MANOLO'S BARBERSHOP PROJECT


PROJECT DESCRIPTION


This project entailed converting a vacant rental space in a commercial building into an elegant and contemporary barbershop.
We added interior walls to establish the barbershop arrangement and constructed a new washroom in the back. Modern mirrors
and shelving for barbers and clients were integrated into the design, along with steel mesh backing for a secure and appealing appearance.
The room was decorated with exquisite granite floor tiles, which added to the shop's overall charm. We also set aside areas for barber stations to ensure
functionality and comfort. The end result is an entirely operational barbershop that blends modern style with practical elements, satisfying the client's vision and operating requirements.
